WPFZooManager

Назначение программы

WPFZooManager, это приложение для управления зоологическим парком, разработанное с использованием технологий Windows Presentation Foundation (WPF) и языка C#. Программа позволяет просматривать, добавлять, обновлять и удалять зоопарки и животных, а также устанавливать связи между ними. Это решение предназначено для удобного управления данными о зоопарках и их питомцах в базе данных SQL Server.

Реализованные функции:

- Просмотр списка зоопарков и их расположений.
- Просмотр всех животных и их имен.
- Добавление новых зоопарков и животных.
- Обновление данных о зоопарках и животных.
- Удаление зоопарков и животных из базы данных.
- Назначение животных на определенные зоопарки.
- Просмотр животных, связанных с выбранным зоопарком.

Как запустить проект?

1. Требования:
   - Visual Studio (поддержка C# и WPF)
   - SQL Server (местный или удаленный) с настроенной базой данных
   - Конфигурационный файл `App.config` с правильной строкой подключения к базе данных (используется `AnimalZooDBConnectionString`)

2. Настройка базы данных:
   - Создайте базу данных, которая содержит таблицы: `Zoo`, `Animal`, и связующую таблицу `ZooAnimal`.
   - Обеспечьте наличие необходимых данных в таблицах или добавьте их вручную.

3. Настройка проекта:
   - Откройте решение в Visual Studio.
   - Проверьте, что строка подключения в файле `App.config` соответствует вашей базе данных.

4. Запуск проекта:
   - Соберите решение.
   - Нажмите кнопку "Запуск" или используйте F5 для запуска приложения.
   - В интерфейсе выберите, добавляйте, редактируйте или удаляйте зоопарки и животных по необходимости.
